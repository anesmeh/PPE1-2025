Mini-projet — Analyse d'URLs

Ce site présente le travail réalisé dans le cadre du mini-projet PPE1. Un script Bash analyse une liste d’URLs et génère automatiquement un tableau contenant :

  • le numéro de ligne
  • l’URL correspondante
  • le code HTTP de la réponse
  • l’encodage détecté
  • le nombre de mots sur la page

Le tableau est automatiquement généré dans le dossier tableaux/ à partir du fichier d’URLs urls/fr.txt.
